# E-Commerce Chatbot

## Description
This project implements a rule-based e-commerce chatbot that assists customers with product search, order tracking, and return policy queries. The chatbot uses Natural Language Processing techniques to preprocess user input, detect intent, and provide context-aware responses.

---

## Problem Statement
E-commerce platforms receive repetitive customer queries regarding product availability, order status, and return policies. Automating these interactions improves customer support efficiency and user experience.

---

## Objective
- Accept customer e-commerce queries  
- Preprocess user input using NLP techniques  
- Detect customer intent from text  
- Provide relevant responses for common support scenarios  

---

## Methodology
1. User enters a query  
2. Text is cleaned and tokenized  
3. Keywords are matched against predefined intent categories  
4. Intent is identified using rule-based classification  
5. Product database / predefined responses are used to generate reply  

---

## NLP Techniques Used
- Text Lowercasing  
- Punctuation Removal  
- Tokenization  
- Keyword-Based Intent Matching  
- Rule-Based Response Generation  

---

## Technologies Used
